### Pong en tu compu
Este proyecto es un juego de pong para jugar en un browser con dos jugadores.

El proyecto esta en base a HTML, CSS y JavaScript y funciona de manera local.

#### Estructura de archivos recomendada

- `index.html`: Archivo principal con la estructura del juego y el canvas.
- `style.css`: Archivo de estilos para el diseño visual del juego.
- `pong.js`: Lógica principal del juego (movimiento de paletas, pelota, colisiones y marcador).
- `README.md`: Documentación y guía de uso.

Puedes agregar otros archivos si lo necesitas, pero estos son los esenciales para el MVP.
